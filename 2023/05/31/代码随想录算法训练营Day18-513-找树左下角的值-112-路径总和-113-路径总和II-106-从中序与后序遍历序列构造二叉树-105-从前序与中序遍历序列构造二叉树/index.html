112.路径总和 113.路径总和II

题目：给你二叉树的根节点 root 和一个表示目标和的整数 targetSum 。判断该树中是否存在 根节点到叶子节点 的路径，这条路径上所有节点值相加等于目标和 targetSum 。如果存在，返回 true ；否则，返回 false 。

112.可视化

分析：

  1. 用前序遍历的方法
  2. 思路是把 targetsum 用 count 计数，从根节点开始递归往下减，看到最后的叶子节点值是否为0。如果为0，再递归回去为给定的targetsum。
