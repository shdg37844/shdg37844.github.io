1.两数之和

分析：

  1. 用哈希表。牺牲空间换时间。时间复杂度为O(n)，空间复杂度是O(n)。

  2. 用双指针法做。双指针会用到排序，由于排序的时间复杂度为 O(nlogn)，所以整体的时间复杂度为 O(nlogn)。其次，排序后会打乱原始数组的顺序，通用的解决办法是利用一个Pair来存储value和index的对应关系，然后对Pair的value进行排序。

    ```js
    var twoSum = function(nums, target) {
    // 创建一个 Pair 数组来保存元素和索引的对应关系
    let pairs = nums.map((num, index) => ({ num, index }));

    // 根据元素的值对 Pair 数组进行排序
    pairs.sort((a, b) => a.num - b.num);

    let lo = 0;
    let hi = nums.length - 1;

    while (lo < hi) {
    let sum = pairs[lo].num + pairs[hi].num;
    if (sum < target) {
    lo++;
    } else if (sum > target) {
    hi–;
    } else {
    // 找到符合条件的元素，返回它们的索引
    return [pairs[lo].index, pairs[hi].index];
    }
    }

    // 没有找到符合条件的元素
    return [];
    };
